## System Requirements

To use **Documind**, ensure your system meets the following requirements:

- **Node.js**: Version 16 or later
- **npm**: For package management
- **Ghostscript**: Installed for PDF processing  
  - [Installation guide for Ghostscript](https://ghostscript.com/releases/index.html)
- **GraphicsMagick**: For image processing
  - [Installation guide for GraphicsMagick](http://www.graphicsmagick.org/download.html)


## Install the Package

To install **Documind**, run the following command using your preferred package manager:

```bash npm
npm install documind
```

## Environment Variables

Set up your `.env` file.

```bash .env.example
OPENAI_API_KEY=<your-openai-api-key>
```

| Key                  | Description                                                                 |
|----------------------|-----------------------------------------------------------------------------|
| **OPENAI_API_KEY**   | Your OpenAI API key.                       |
